# Mode

insert mode : i

# Navigation

- left : h 
- right : l 
- top : k 
- bottom : j
- word by word : w
- word by word back : b
- end of a word no space : e
- back to beginning : ^ 
- go to the end : $
- undu : u 
- redo : Ctrl + r
- delete right : x
- delete left : X
- go to the end of a parenthesis:%
- delete all line : D or dd
- go to paragraph above : + 
- got to paragraph under : -
- cut : dd 
- paste : p 
- paste before cursor : P
