# Unix
## Symbolic link
### List  
```ls -la ```
## Path
### Back to previous path
```cd -```
### Go to home directory
```
cd 
cd ~
```
## File
### Locate file
```find / -name file.txt```
### Copy
```
cp fileToCopy newNameFile
cp /directory/file.txt /directory/file.txt
cp /home/name/file.txt /home/myPath
```
### Remove
```
rm filename
```
